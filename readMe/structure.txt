구조체 (structure)
다양한 데이터를 한꺼번에 처리하기 위해서 사용

사실 구조체는 자바에서 사용되는 개념은 아니다.
자바에서는 구조체로 class를 이용한다. 
---------------------------------------
객체지향 프로그래밍
문제를 해결함에 있어서 문제해결 순서에 집중하지 않고
문제를 해결하는 객체에 집중한다.

객체는 어떻게 만들어내나?
>>클래스를 이용해서 생성

객체는 상태값(데이터)과 동작(메서드)을 가진다.
클래스로 부터 여러 객체를 만들어낼 수 있고
각 객체들은 각각의 데이터와 메서드를 가진다.

로또 프로그램을 객체지향으로 바꿔보자.
객체지향 : 실제 세계를 모델링 == 문제를 해결하는 객체를 만들어 냄
로또 프로그램에서 문제는 중복되지 않는 6개의 난수를 생성해야 하는 것
난수를 생성하는 객체를 만들자.
