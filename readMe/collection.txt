Collection 사용법에 대해 알아보자.
Map, Set

Set : 중복을 허용하지 않는 자료구조
Map : key-value 형태의 자료구조
key(문자열): value(학생)
1 		 : 홍길동
2 		 : 신사임당
3		 : 김장금
5		 : 이순신
7		 : 이순신 //key는 중복이 되지않는 값이 들어가며, 밸류는 중복되어도 상관없다.
"Hong"	 : 홍길동 //문자열 가능.

구조
Map<타입, 클래스> {

}
타입에는 보통 스트링을 넣지만 다른걸 넣어도 상관은 없음.
클래스의 경우 여러종류의 클래스를 담고 싶으면 Object로 넣으면 된다.
왜? 오브젝트는 모든 클래스가 상속해서 사용하기때문에!
